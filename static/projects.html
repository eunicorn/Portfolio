Projects

PEACH

Peach is an interactive data visualization app that helps restaurant owners to determine the most profitable location in San Diego based on their budget for rent, target age for consumers, and crime rates.

ÜBERÜNO

ÜBERÜNO shows where Uber/Lyft drivers can gain the most profit by identifying areas in San Diego with the least availability of vehicles. It displays the result in an interactive data visualization manner.

GRAPE

Grape is a mobile web app that allows users to document all the good deeds that they have done so they can make time and strive to create a positive environment. The goal of this app is to create more incentive for individuals to perform good deeds.

RETROBIKE

RetroBike is an indoor bike that powers a gaming emulator in order to make fitness more exciting for children. To ensure safety, I created a more interactive and engaging user tutorial that thoroughly explains setup and safety features.

Global Ties & One Step

Worked with a nonprofit organization to create and design a secure survey application that will be used by clients in Nepal.

DrawBot

A robot that could write and draw various shapes and letters.
